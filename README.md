# 📊 US Equity Systematic Trend Strategy
## 美股系统化多因子趋势策略（实时记录）

---

## 🎯 概览

本项目用于展示一套可执行的美股系统化多因子趋势策略在真实市场环境下的持续验证结果。

策略以 **15分钟周期趋势建模** 为核心，结合动量与波动过滤因子，在高流动性美股标的中进行信号驱动交易。

- **执行方式**：API信号驱动执行
- **交易模式**：纯信号输出（资金与风控权限由资方完全掌控，非源码出售）
- **数据延迟**：约 1–30 分钟
- **交易频率**：约 1–5 笔/交易日
- **主要标的**：高流动性美股龙头（以 TSLA 等标的为主要验证对象）

## 🛡 风控体系（多层熔断）

系统内置多级风险控制机制：

1. 单笔风险控制：-3% 强制止损并暂停当日交易
2. 日内风险控制：-5% 日内累计亏损熔断
3. 全局回撤控制：-10% 系统级强制锁定

> 风控优先级：全局 > 日内 > 单笔
> 风控机制可与券商端风控系统双重对接执行
